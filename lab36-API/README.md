# API : Application Programming Interface
API is a medium or interface between two programming languages and two different systems.
API used to share data from one plateform to another plateform or from one service to another service.
API is used as an Abstraction layer, and used to use service from another plateforms to avoid implementation.

## What is FastAPI

``FastAPI`` is a webframe API build for python backend,
fastapi follows standars of ``openAPI`` or formally ``Swagger``

### Key features fo FastAPI
1. fastAPI is opensource web framework for building APIs in python.
2. fast api is easy to learn, easy to implement and used to build production ready modern websites.


you can download ``fastapi`` in a virtual environment . use this command in terminal or CMD 
: ``pip install fastapi`` or ``pip install fastapi[standarad]`` for standard tools with preinstalled dependencies.


### FastAPI made from what ?
fastapi is made on top of ``starletter`` and ``pydantic``
