## While-loop
***objective :*** whileloop used when we do not know the total number of iterations or simply if we don't know how  many times a statement or code block run.

- while-loop executes block of codes depends on conditon


### syntax of data 

``  while ( <condition>) ``
### program to print number from 1 to 10

```python
i = 1
while ( i <=10) :
    print(i)
    i =  i + 1
```
### program to print charecters of a string 
```python
name = "Welcome AT "
for i in name :
    print(i)

```
